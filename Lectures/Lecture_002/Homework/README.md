### Question 5: Find the factorial of a given number.

#### Flowchart:
![Q6](https://via.placeholder.com/468x300?text=App+Screenshot+Here)

#### Pseudocode:
- START
- READ n
- IF (N == 0) THEN 
  - WRITE "ZERO"
  - END
- ELSE
  - fact = 1
  - count = 2
  - WHILE (count <= n) DO
    - fact = fact * count
    - count = count + 1
  -  END WHILE
  - WRITE fact
- END
